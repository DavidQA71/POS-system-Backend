# POS System

Sistema de Punto de Venta (POS) hecho con **Node.js**, **Express**, **TypeScript** y **MySQL**.

## 🚀 Instalación

### Clonar el repositorio
```
git clone https://github.com/usuario/pos-system.git

```

### Entrar al proyecto
```
cd pos-system

```

### Instalar dependencias
```
npm install

```

### **Crear un archivo .env con:**
```
PORT=3000
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=123456
DB_NAME=pos_system
JWT_SECRET=secret
```

### Ejecutar:
```
npm run dev

```
